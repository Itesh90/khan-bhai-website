## 🔐 Security Layers

### Payment Security
- HMAC-SHA256 signature verification
- Razorpay handles PCI DSS (no card data on server)
- Test mode → live mode seamless transition
- Rate limiting: 10 requests/IP/hour

### Authentication
- NextAuth.js + JWT (24-hour expiry)
- HttpOnly secure cookies
- Middleware protection (/admin/*)
- Bcrypt password hashing

### Data Security
- API keys in .env only (server-side)
- Zod input validation
- Prisma parameterized queries
- HTTPS enforced everywhere
